Car Racing Game with Genetic Algorithm

An interactive car racing game where you can race manually or train AI cars using a Genetic Algorithm. Compete with AI opponents, complete laps, and watch AI improve over generations.

Features
Manual race mode (WASD / Arrow keys)
AI racing using Genetic Algorithm
Dynamic checkpoints and lap tracking
Responsive design for desktop & mobile
How to Play
Open index.html in a browser.
Select number of competitors, laps, and mode.
Click Start Race and drive or watch AI race.
Controls
W / ↑: Accelerate
S / ↓: Brake / Reverse
A / ←: Turn Left
D / →: Turn Right
Space: Hand Brake
R: Reset Car
Enter: Start Race
